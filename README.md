## Описание
Java Core проект, демонстрирующий работу с реляционной базой данных MySQL с использованием JDBC.
Приложение реализует CRUD-операции над сущностью User с разделением ответственности по слоям (DAO / Service), частичной реализацией MVC и покрытием логики JUnit-тестами.

## Используемые технологии
- Java (Core)
- JDBC
- MySQL
- Maven
- JUnit
- MVC (DAO / Service / Model)
